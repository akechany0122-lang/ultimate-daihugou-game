# Ultimate Daihugou (究極の大富豪)

ダークファンタジーな世界観の対戦型大富豪ゲームです。

## 遊び方
1. `index.html` をブラウザで開きます。
2. タイトル画面で「生贄の宴 (GAME START)」を押すとゲームが始まります。
3. 自分の番が来たら、場に出ているカードより強いカードを選択して「出す」ボタンを押します。
4. 出せるカードがない、または出したくない場合は「パス」を押します。
5. 全員が上がった（手札がなくなった）ら、順位に応じて次回のカード交換が発生します。

## 公開方法（ホスティング）
このゲームは静的な HTML/CSS/JavaScript で構成されているため、以下のサービスで簡単に公開できます。

### GitHub Pages
1. GitHub にリポジトリを作成し、このフォルダのファイルをプッシュします。
2. リポジトリの Settings > Pages から、公開するブランチ（main等）を選択します。
3. 数分後に `https://<ユーザー名>.github.io/<リポジトリ名>/` で公開されます。

### Vercel / Netlify
1. 各サービスにログインし、「New Project」から GitHub リポジトリを連携、またはフォルダをドラッグ＆ドロップします。
2. 設定はそのままでデプロイを実行します。

## ファイル構成
- `index.html`: ゲームのメインロジック、UI、スタイルが含まれます。
- `*.webp`: カードやロゴに使用される画像ファイル。
- `bgm.mp3`, `bgm2.mp3`: ゲーム内の音楽。
